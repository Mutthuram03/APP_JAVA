🛠 How to Use

Clone this repository:

git clone https://github.com/Mutthuram03/APP_JAVA.git


Navigate into the project folder:

cd APP_JAVA


Open files in your favorite IDE (VS Code, IntelliJ IDEA, or Eclipse).

Compile and run the Java files:

javac FileName.java  
java FileName
